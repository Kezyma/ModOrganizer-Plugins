OpenMW Player

OpenMW Player manages OpenMW configuration files and gameplay settings for Morrowind modding through Mod Organizer 2.

Features

  • Settings Management: Edit OpenMW settings organized by category
  • Archive Control: Enable, disable, and reorder game archives (.bsa files)
  • Groundcover Support: Manage groundcover plugins with sorting options
  • Config Editing: Edit openmw.cfg entries with a structured interface
  • Raw Editor: Direct text editing of settings.cfg and openmw.cfg
  • Import/Export: Backup and restore configurations

Tabs

Options

Configure the path to your openmw.cfg file and enable support for .omwaddon and .omwscripts files through dummy .esp creation.

Archives

Manage game archives (.bsa files). Use checkboxes to enable or disable archives, and drag to reorder them. The order determines which archive takes priority when files conflict.

Groundcover

Filter and manage groundcover plugins. Sort options include:

  • Load Order: Sort by current load order
  • A-Z: Alphabetical ascending
  • Z-A: Alphabetical descending

Settings

Edit OpenMW settings organized by category (Camera, Terrain, GUI, etc.). Each setting shows its current value and can be modified directly. Changes are saved to settings.cfg.

Openmw.cfg

Edit custom openmw.cfg entries organized by category tabs. Add, modify, or remove configuration entries that control OpenMW behavior.

Raw Editor

Access the text editor to manually edit settings.cfg and openmw.cfg files directly. Use this for advanced editing or troubleshooting.

Settings

  • openmwcfgpath: Path to your openmw.cfg file
  • dummyesp: Enable dummy .esp file creation for .omwaddon and .omwscripts support

Troubleshooting

  • Settings not saving: Ensure the openmw.cfg path is correctly configured in the Options tab.
  • Archives not loading: Verify the archive order and ensure archives are enabled in both OpenMW Player and your mod list.
  • Groundcover not appearing: Check that groundcover plugins are enabled and properly sorted.
