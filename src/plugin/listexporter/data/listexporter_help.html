List Exporter

Export your mod list to various formats for sharing or documentation.

Features

  • Multiple Formats: Export to HTML, Markdown, CSV, or JSON
  • Multi-Profile: Export mods from multiple profiles at once
  • Category Separation: Optionally create separate tables for each category
  • Column Selection: Choose which information to include and reorder columns
  • Nexus Links: Mod names link to NexusMods when mod ID is available
  • Exclusion Tag: Exclude specific mods from export using a tag

Usage

  1. Select the export format (HTML, Markdown, CSV, or JSON)
  2. Choose whether to separate tables by category
  3. Select one or more profiles to include
  4. Choose which columns to include in the export
  5. Click Export and choose a save location

Columns

  • Mod Name: The name of the mod, hyperlinked to NexusMods if a mod ID is available
  • Category: The separator/category the mod belongs to (when not using separate tables)
  • Notes: Notes from the mod's meta.ini file
  • Comments: Comments from the mod's meta.ini file
  • Profile columns: When multiple profiles are selected, shows enabled status per profile

Excluding Mods

To exclude a mod from exports, add <LE_Exclude> anywhere in the mod's Notes or Comments field in Mod Organizer. The tag is not case-sensitive.

This is useful for hiding utility mods, merged patches, or other mods you don't want to appear in shared mod lists.

Export Formats

  • HTML: Styled web page with Bootstrap, ideal for sharing online
  • Markdown: Plain text tables, ideal for GitHub, Discord, or forums
  • CSV: Spreadsheet-compatible format for Excel or Google Sheets
  • JSON: Structured data format for programmatic use

Notes

  • Mods are included if they are enabled in any selected profile
  • When multiple profiles are selected, profile columns show checkmarks or crosses
  • Categories are determined by separator mods in your mod list
  • Mod information is read from each mod's meta.ini file
  • Columns can be reordered by dragging them in the column list

Troubleshooting

Mod names don't have Nexus links.

Nexus links are only added when a mod has a valid mod ID in its meta.ini file. Mods installed manually or from non-Nexus sources won't have this information.

Categories are wrong or missing.

Categories are determined by separator mods (mods ending in "_separator"). Ensure your separators are properly named and positioned in your mod list.

Some mods are missing from export.

Check if the missing mods have <LE_Exclude> in their Notes or Comments. Also verify that the mods are enabled in at least one of the selected profiles.
