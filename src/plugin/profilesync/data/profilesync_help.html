Profile Sync

Profile Sync allows you to synchronize mod configurations across multiple Mod Organizer 2 profiles, keeping them in sync automatically or creating profile groups with different mod loadouts.

Features

  • Sync Groups: Create groups of profiles that stay synchronized
  • State Groups: Define different states within a group for varied configurations
  • Automatic Sync: Changes propagate automatically when mods are installed, removed, enabled, disabled, or reordered
  • Category Control: Choose which mod categories to sync for each state

Concepts

Sync Groups

A sync group is a collection of profiles that share the same mod configuration. When you make changes to one profile in a sync group, those changes are automatically applied to all other profiles in the group.

State Groups

State groups allow you to create different configurations within a sync group. Each state can include different profiles and mod categories, giving you fine-grained control over what gets synchronized.

Usage

Creating a Sync Group

  1. Open the Profile Sync dialog from the Tools menu
  2. In the Sync Groups tab, click to add a new group
  3. Select which profiles should be part of this group
  4. Changes made in any profile will now sync to others in the group

Creating State Groups

  1. Switch to the State Groups tab
  2. Add a new state group
  3. Select which profiles belong to this state
  4. Choose which mod categories should be synchronized for this state

Managing Groups

Use the interface to add, remove, and modify sync groups and state groups. You can have multiple groups with different configurations to suit various modding scenarios.

Troubleshooting

The last change made is not synchronizing to other profiles.

Profile Sync runs on specific events: when a mod is installed, removed, enabled/disabled, or reordered through drag and drop. If another action causes a change, it may not trigger a sync.

To force a sync, toggle the enabled state of any mod back and forth once.

Sync is slow or causes lag.

Profile Sync can use threaded operations for better performance. Check the plugin settings for async options.

Some mods are not syncing.

Verify that the mod's category is included in your state group configuration. Only categories selected for a state will be synchronized.
