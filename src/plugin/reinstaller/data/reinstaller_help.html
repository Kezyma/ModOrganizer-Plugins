Reinstaller

Reinstaller allows you to create quick installers from downloaded mods and re-run them without re-downloading. This is useful for mods with complex FOMOD installers that you may want to reconfigure.

Features

  • Quick Installers: Save mod installers for easy re-running
  • No Re-download: Re-install mods without downloading again
  • Multiple Versions: Keep different versions of the same mod installer
  • FOMOD Support: Re-run FOMOD installers to change options

Usage

Creating an Installer

  1. Open the Reinstaller dialog from the Tools menu
  2. Select a downloaded mod from the dropdown list
  3. Enter a name for the installer (or use the default)
  4. Click Add to create the installer

Running an Installer

  1. Select the installer from the list
  2. If multiple versions exist, select the desired version from the dropdown
  3. Click Install to run the installer
  4. Complete the installation process (FOMOD options, etc.)

Managing Installers

  • Delete: Remove an installer you no longer need
  • Version Selection: Choose between different versions of the same mod

Tips

  • Create installers for mods with complex FOMOD options that you might want to change later
  • Keep installers for mods you frequently reinstall or update
  • The installer uses the original downloaded archive, so don't delete it from your downloads folder

Troubleshooting

Installer not running.

Ensure the original download file still exists in your Mod Organizer downloads folder. If the file has been deleted, you'll need to re-download the mod.

Wrong version installing.

Check the version dropdown to ensure you have the correct version selected before clicking Install.

Mod not appearing in download list.

Only mods in your Mod Organizer downloads folder will appear. Ensure the mod has been downloaded through Mod Organizer.
