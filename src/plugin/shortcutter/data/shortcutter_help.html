Shortcutter

Shortcutter creates desktop shortcuts that launch Mod Organizer 2 with a specific profile and application. Launch your modded games directly from the desktop without opening Mod Organizer first.

Features

  • Desktop Shortcuts: Create shortcuts on your desktop
  • Profile Selection: Launch with a specific MO2 profile
  • Application Selection: Choose which application to run
  • Custom Icons: Use custom icons for your shortcuts
  • Auto-naming: Shortcuts are automatically named based on profile and application

Usage

Creating a Shortcut

  1. Open the Shortcutter dialog from the Tools menu
  2. Select the application you want to launch from the dropdown
  3. Select the profile you want to use
  4. Optionally customize the shortcut name
  5. Optionally browse for a custom icon (.ico or .exe file)
  6. Click Create to generate the shortcut

Shortcut Name

By default, shortcuts are named "Profile - Application". You can edit this to any name you prefer.

Custom Icons

Click the browse button to select a custom icon. You can use:

  • .ico files for direct icon use
  • .exe files to extract their embedded icon

A preview of the selected icon is shown before creating the shortcut.

How It Works

Shortcuts created by Shortcutter launch Mod Organizer with command-line parameters that specify the profile and application to run. When you use the shortcut, MO2 opens with the selected profile active and immediately launches the chosen application.

Troubleshooting

Where is my shortcut?

All generated shortcuts are placed on the desktop, or wherever Windows believes your desktop is located.

Opening Mod Organizer defaults to the last instance that Shortcutter opened.

The current Mod Organizer instance is set in the Windows registry. When Shortcutter launches from a shortcut, the instance in the registry is changed. Unfortunately, there is no way to change it back automatically, so this must be done manually if you want to switch instances.

Shortcut doesn't work.

Ensure that:

  • The profile still exists in Mod Organizer
  • The application still exists in the profile's executable list
  • Mod Organizer hasn't been moved to a different location

Icon not showing correctly.

Windows caches shortcut icons. Try refreshing your desktop or restarting Explorer if a new icon doesn't appear immediately.
